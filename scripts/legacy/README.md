# Legacy Scripts

These scripts are from early development and are **not the recommended way to mine**.

For mining, use the `skill/` directory instead:
```bash
cp -r skill ~/.openclaw/workspace/skills/clawchain-miner
cd ~/.openclaw/workspace/skills/clawchain-miner
python3 scripts/setup.py
python3 scripts/mine.py
```

These legacy scripts are kept for reference only.
